Descrição:

Este é um projeto voluntário aplicado em uma barbearia real, com dados verídicos.

  • Necessidade do cliente: visualizar métricas anuais de forma unificada e personalizada. Sua aplicação apresentava métricas trimestrais separadamente.
  • Processo de resolução do problema: download dos dados (xlsx), conversão para csv, tratamento dos dados com PySpark (visando treino) e Análise dos dados com PowerBi.
  • Experiência relevante adquirida: comunicação direta com o cliente e construção da análise a partir das suas necessidades, implicando diretamente na construção do código PySpark. Emprego de DAX para definir métricas de fluxo de clientes, como clientes novos, antigos, perdidos, etc.
  • Principal dificuldade: a automatização do processo é possível utilizando Selenium, mas não por API, pois a aplicação não possui.
  • Conclusão: O cliente pode agora visualizar as principais métricas de forma conjunta, filtrando as informações em um único dashboard, por ano, trimestre, mês, serviço e profissional.
